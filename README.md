# KIF Mega Stones
**Mega Stones for your Infinite Fusion!**

## Features
- Adds Mega Stones to the game!
- If a species with a mega stone gets it to hold, it gets stat/ability/type changes.
- Fusion support: Fusions containing a species with a mega stone also get these changes.
- Fusion mega stone type changes will be applied according to head/body, the ability will always apply.
- How to gain stones: When you fuse two of the same species, they will hold the mega stone after fusing.
- Species with multiple stones will get the other stones added to the player's inventory.

**KIF Version: 0.20.7**
**Script Version: 1.1.0**
